freeCodeCamp Project: URL Shortener Microservice

User stories:

  1. I can pass a URL as a parameter and I will receive a 'shortened' URL in the JSON response.
  2. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  3. When I visit that shortened URL, it will redirect me to my original link.

Example of generating a short-cut URL

Entering:

      https://url-shortener-microservice-pa.glitch.me/new/https://www.bbc.co.uk/weather/0/sp3

will give a response similar to:

      {
          "original_url": "/https://www.bbc.co.uk/weather/0/sp3",
          "short_url": "https://url-shortener-microservice-pa.glitch.me/1433"}
      }

Example of using a short-cut URL

Entering:

      https://url-shortener-microservice-pa.glitch.me/1433

will redirect to:

      /https://www.bbc.co.uk/weather/0/sp3
